# Getting Started

#### This application needs
- Maven <= 3.5
- Java 11

#### Also this application implements
- R2DBC
- Spring Webflux

### Compile application
```
mvn clean compile
```

### Test application
```
mvn test
```

### Run application
```
java -jar <jarname>.jar
```
